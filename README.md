## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/nrrb/number-guessing-game.git
```

2. From a command line where Python is available, change directory to the repo and run:

```bash
python task-cli.py
```

## Running Tests

Run:

```bash
python3 -m unittest discover tests
```
